# ML Learning Pack — Andrew Ng Machine Learning Specialization через практику

Полный самоучитель по ML на русском языке: 18 Jupyter-ноутбуков, 5 учебных проектов,
2 портфолио-проекта, шпаргалка, 50 задач и 50 промптов для обучения с AI.

## Как пользоваться

1. Открой `roadmap_8_weeks.md` — это твой план. Иди по неделям.
2. Каждый урок — готовый ноутбук в `notebooks/`. Открывай в Jupyter / Codespaces,
   читай markdown-ячейки, запускай код, делай мини-задания прямо в ноутбуке.
3. В конце каждого урока — 5 проверочных вопросов и домашнее задание. Не пропускай!
4. После блока уроков делай проект из `projects/` — он закрепляет тему.
5. Недели 7–8 — портфолио-проекты из `portfolio/`. Их выкладывай на GitHub.
6. Если что-то непонятно — открой `prompts_for_future.md`, там готовые промпты,
   чтобы спросить любую AI-модель (включая бесплатные).
7. `ml_cheatsheet.md` держи открытой рядом — это словарь всех терминов.
8. `practice_tasks.md` — решай по 2–3 задачи в день для тренировки.
