# 50 промптов для обучения ML с любым AI

Подставляй свои темы/код в [скобки]. Работают с любой моделью, включая бесплатные.

## Объяснение тем (1–10)

1. "Объясни [тему] так, как будто мне 12 лет, без формул. Потом объясни ещё раз на уровне студента. Потом покажи минимальный Python-пример."
2. "Объясни [термин] через аналогию из реальной жизни. Дай 3 разные аналогии и скажи, где каждая аналогия ломается."
3. "Я путаю [понятие А] и [понятие Б]. Сделай таблицу различий и дай тест из 5 вопросов, чтобы проверить, что я понял."
4. "Объясни формулу [формула] по частям: что значит каждый символ, что формула делает по смыслу, и приведи числовой пример с подстановкой."
5. "Какие 3 самых частых заблуждения новичков про [тему]? Объясни, почему они неверны."
6. "Объясни [тему] через пошаговый числовой пример с маленькими числами, который я могу пересчитать на бумаге."
7. "Расскажи, где [тема/алгоритм] применяется в реальных продуктах (банки, e-commerce, медицина). 5 конкретных примеров."
8. "Сравни [алгоритм А] и [алгоритм Б]: когда выбирать какой, плюсы/минусы, и дай код-пример обоих на одних данных."
9. "Я прочитал про [тему], проверь моё понимание. Вот как я её понимаю своими словами: [твоё объяснение]. Найди ошибки и неточности."
10. "Составь мне план изучения [темы] на 5 дней по 1 часу с практикой в Jupyter каждый день."

## Разбор кода (11–20)

11. "Объясни этот код построчно для новичка. Для каждой строки: что делает, зачем нужна, что будет, если её убрать: [код]"
12. "У меня ошибка: [текст ошибки]. Вот код: [код]. Объясни причину простыми словами, исправь и скажи, как избегать этой ошибки в будущем."
13. "Вот мой код: [код]. Он работает, но проверь: есть ли ошибки в ML-логике (data leakage, неправильный split, утечка test в обучение)?"
14. "Перепиши этот код с подробными русскими комментариями к каждой строке: [код]"
15. "Вот код из курса Andrew Ng: [код]. Объясни, что здесь X, что y, какие shape у массивов на каждом шаге, и что подаётся в модель."
16. "Упрости этот код, не меняя результат, и объясни, что ты изменил и почему: [код]"
17. "Этот код использует циклы: [код]. Перепиши через векторизацию NumPy и объясни, почему так быстрее."
18. "Я написал [алгоритм] с нуля: [код]. Сравни с реализацией sklearn: правильно ли я его понял, что упростил, что упустил?"
19. "Добавь в этот код проверки данных: shape, dtype, пропуски, и объясни, зачем каждая проверка: [код]"
20. "Объясни, что выведет этот код, ДО того как я его запущу, шаг за шагом: [код]"

## Создание notebook-уроков (21–30)

21. "Создай Jupyter-урок по теме [тема] по ячейкам: Markdown-объяснение → код → построчный разбор → визуализация → типичные ошибки → конспект → мини-задание → 5 проверочных вопросов → домашка. Уровень: новичок, объясняй просто, на русском."
22. "Создай notebook-практикум: 5 упражнений по [теме] от простого к сложному, с пустыми ячейками для моего кода и скрытыми ответами в конце."
23. "Создай notebook, где [алгоритм] реализован дважды: вручную на NumPy и через sklearn, с сравнением результатов."
24. "Создай notebook-эксперимент: как параметр [параметр] влияет на [алгоритм]. Перебери 5 значений, построй графики, сделай выводы."
25. "Сгенерируй реалистичный синтетический датасет для задачи [задача] (200 строк, 5 признаков, с шумом и парой выбросов) и notebook для его исследования."
26. "Создай notebook 'разбор ошибок': покажи 5 типичных багов новичков в [теме], каждый — сломанный код, текст ошибки, объяснение, исправление."
27. "Создай продолжение урока: я уже знаю [список тем]. Следующая тема — [тема]. Свяжи её с тем, что я знаю."
28. "Сделай notebook-шпаргалку по [библиотеке]: 20 самых нужных команд с примерами и комментариями."
29. "Создай notebook с визуализациями, объясняющими [тему] (например, как gradient descent спускается по cost-функции), с анимацией или серией графиков."
30. "Преобразуй мой конспект в интерактивный notebook с кодом для каждого пункта: [конспект]"

## Проверка проектов (31–38)

31. "Вот мой ML-проект: [код/описание]. Проведи code review как senior ML-инженер: логика, структура, метрики, что улучшить. Будь строгим, но конструктивным."
32. "Проверь мой проект на типичные ошибки: data leakage, обучение на test, отсутствие масштабирования, неправильные метрики: [код]"
33. "Мой проект выдаёт метрики: [метрики]. Это хорошо или плохо для задачи [задача]? С чем сравнивать? Что такое baseline и какой он здесь?"
34. "Какие 5 улучшений добавить в этот проект, чтобы он выглядел сильнее в портфолио? Отсортируй по соотношению польза/трудозатраты: [описание проекта]"
35. "Помоги сделать error analysis: вот предсказания и реальные значения [данные]. На каких примерах модель ошибается сильнее и почему это может быть?"
36. "Преврати мой notebook в структуру проекта для GitHub: какие файлы создать (src/, README, requirements), что куда вынести: [описание]"
37. "Задай мне 10 вопросов про мой проект, как на собеседовании junior ML-инженера: [описание проекта]. Потом оцени мои ответы."
38. "Сравни мой проект с тем, что обычно ждут от junior-портфолио. Чего не хватает? [описание]"

## GitHub и README (39–43)

39. "Напиши README.md для ML-проекта: [описание]. Структура: что делает проект, данные, как запустить, результаты с метриками, структура папок, идеи улучшения. Стиль — как у хороших open-source проектов."
40. "Напиши короткое GitHub description (1 строка) и 5 topics-тегов для проекта: [описание]"
41. "Проверь мой README и сделай его профессиональнее: [текст]"
42. "Составь requirements.txt для проекта, использующего: [список библиотек]. Объясни, нужно ли фиксировать версии."
43. "Напиши секцию 'Results' для README: вот мои метрики [метрики] и графики [описание]. Сформулируй выводы честно, без преувеличений."

## Computer Vision — следующий шаг (44–50)

44. "Я знаю основы ML (regression, classification, простые нейросети на Keras). Составь roadmap входа в Computer Vision на 4 недели с практикой."
45. "Объясни, что такое свёрточная нейросеть (CNN) простыми словами: зачем свёртки, что такое фильтр, pooling. Без тяжёлой математики, с аналогиями."
46. "Создай notebook: классификация изображений CIFAR-10 на Keras с простой CNN, с построчным объяснением для новичка."
47. "Объясни разницу между задачами CV: classification, detection, segmentation. По одному реальному примеру и одной популярной модели на каждую."
48. "Покажи, как загрузить предобученную модель (transfer learning) в Keras и дообучить на своих картинках. Объясни, почему это работает лучше обучения с нуля."
49. "Создай мини-проект CV для портфолио: [идея, например классификатор еды/растений]. Дай структуру, код по ячейкам, README."
50. "Я прошёл [список тем]. Проведи мне устный экзамен из 15 вопросов по возрастающей сложности, оценивай каждый ответ и в конце дай общую оценку и пробелы."
